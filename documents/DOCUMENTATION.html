­Ъћљ ACMAC00 Рђћ authorization-code cipher engine COBOL

The heart of the module: a reversible cipher that packs the machine "fingerprint" (system type, 8-char serial, LPAR number, library name, expiry date, auth type) into a scrambled 30-character authorization code, and reverses it. It performs a fixed column-transposition (the 30 payload characters are scattered to specific positions) combined with a per-character substitution Рђћ a Caesar-style shift (K=26, MOD=30) for letters and a 10├Ќ10 digit-rotation table for digits. Parameter LINK-PARM selects 'E'ncrypt or 'D'ecrypt.

PRRDMSG is a generic error-message retrieval service Рђћ not ACM-specific. Given a 5-character message number it returns the message text in the caller's language (Hebrew or English).
